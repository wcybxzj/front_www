### 第七讲===>组件化开发

- #### 创建组件的两种方式
  - ```
    var Header = { template:'模板' , data是一个函数,methods:功能,components:子组件们 }//局部声明
    ```

    ```
    Vue.component('组件名',组件对象);//全局注册 等于注册加声明了
    ```


- #### 组件类型

  - 通用组件（例如表单、弹窗、布局类等)
  - 业务组件（抽奖、机器分类）
  - 页面组件（单页面开发程序的每个页面的都是一个组件、只完成功能、不复用）
